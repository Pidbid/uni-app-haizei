# uni-app-haizei
一个使用uni-app框架写的海贼王视频播放简易app

## 使用步骤
- 下载文件导入 [Dcloud 官网](http://dcloud.io) 的 ide Hbuilder 
- 默认使用 /data/sortData.js 文件内的数据，如果使用自己的数据接口可以修改 /pages/index.vue 文件129行
- 默认 /pages/index.vue 中视频播放器 video 组件的预览图未填写，如有需要可以修改该文件下第10行
- 开启真机调试或者打包即可
